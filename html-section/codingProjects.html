Python-based projects:

  • Curve-Fitting Application

    Designed with various Python libraries, including Streamlit (for the main app interface), NumPy, Pandas, and Matplotlib, this app incorporates many different functions in one! This app can either take hand-inputted X, Y, and Z values, or it can read a CSV file and calculate several different types of curve fitting. It can do both 2D AND 3D Curve Fitting! Link is here!

    NOTE: After a bit, the app goes to sleep. If you do encounter this, please click on the button shown on the screen and wait for a few moments.
